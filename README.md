🧹 Noor Clean - Professional Cleaning & Pest Control Services Website

A modern, responsive website for cleaning and pest control services in Riyadh, Saudi Arabia. Built with Next.js 16 and optimized for SEO.

✨ Features:
• Mobile-first responsive design with modern UI/UX
• Full SEO optimization (Meta tags, Schema, Sitemap)
• Admin dashboard for managing services & articles
• Dynamic blog system with rich content support
• Auto-generated sitemap.xml & robots.txt
• Structured Data (Schema.org) for rich search results
• Floating WhatsApp & Call buttons for instant contact
• Image optimization (WebP/AVIF) for fast loading

🚀 Tech Stack:
• Next.js 16 (Pages Router + Turbopack)
• React 19
• JavaScript (ES6+)
• Advanced CSS3 with variables & responsive design

📁 Project Structure:
• /pages - Next.js pages (routing)
• /components - Reusable React components
• /api - Backend API routes
• /data - JSON data files (services, articles, settings)
• /public - Static assets (images, icons)
• /styles - Global CSS styles

🔗 Key Routes:
• Home: /
• Services: /#services
• Blog: /articles
• Contact: /contact
• Admin: /admin/login

🛠️ Local Development:
1. Clone: git clone https://github.com/Mohamedradwan28/noor-clean.git
2. Install: npm install
3. Dev: npm run dev
4. Build: npm run build
5. Start: npm start

🌐 Deployment:
• Hosted on: Vercel (Free tier)
• Live URL: https://noor-clean.vercel.app

📊 SEO Features:
• Dynamic meta titles & descriptions per page
• Open Graph & Twitter Cards for social sharing
• JSON-LD Schema markup (LocalBusiness, BlogPosting, Service)
• Clean URLs with trailing slash removal
• Optimized robots.txt for crawlers
• Auto-generated sitemap.xml
